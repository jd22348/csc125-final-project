
# CSC-125-201 Final Project
# Project Description
A quiz program in which the user is presented with candidates from an US presidential general election and they have to guess the year.
# Features
- Dictionary storing: The dictionary stores the election matchup and the year or years that it corrosponds to.
- Custom function 'check_answer' that checks the input to see it matches (i.e. whether it is correct or not)
- For loop that loops over each subject in the dictionary.
- The user inputs the election year that they are guessing 
- Formatted output
